Features
From description to device
One spec, one command, one complete hardware project.
⚡

MHDL Standard

One YAML/JSON file describes your entire hardware project. Git-friendly diffs, team-friendly workflows, AI-friendly structure.

✔

Design Rule Checks

Pin conflicts, I2C collisions, power budget, connection integrity, enclosure fit, mounting alignment — validated on every build.

⚙

Pluggable Backends

Wokwi, Arduino, OpenSCAD, KiCad by default. Swap in MicroPython, CadQuery, FreeCAD, or EasyEDA.

◾

3D Print Ready

Auto-generated parametric enclosures with snap-fit clips, screw mounts, cutouts, ventilation, and embossed labels.

⚗

MCP Native

Works with Claude, GPT, Gemini, and any MCP-compatible AI. Hardware design as a conversation.

⧉

Full Pipeline

Circuit, firmware, PCB, enclosure, BOM, and documentation — all from one command, in under a millisecond.

How It Works
Three steps to hardware
From idea to 3D-printable device in seconds.
1

Describe your hardware

Tell the AI what you want to build in plain English. MeshCue Forge generates a complete MHDL spec with auto-assigned GPIOs, connections, and enclosure cutouts.

2

Build all artifacts

One command generates Wokwi circuits, Arduino firmware, OpenSCAD enclosures, BOM, pinout docs, assembly instructions, and 3D print guides.

3

Simulate, print, assemble

Test in Wokwi, 3D print the case, flash the firmware, screw it together. A real device from a single description.

MCP Tools
Four tools, infinite hardware
Expose these to any AI client via Model Context Protocol.
meshforge-describe
Natural language to MHDL spec. Describe your board, get a buildable hardware definition with auto-assigned pins and connections.
meshforge-build
MHDL to all artifacts. Circuit, firmware, enclosure, BOM, and documentation generated in one call.
meshforge-validate
6 Design Rule Checks run automatically. Pin conflicts, power budget, I2C collisions — nothing ships without passing.
meshforge-iterate
Patch and rebuild. Change the enclosure type, swap the MCU, add a sensor — instant rebuild from the updated spec.
